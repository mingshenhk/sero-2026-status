SERO 2026 Unofficial Status Page

This site exists to answer one question honestly: Is SERO still usable in 2026?

Based on current manual testing, the answer is: yes, but unevenly. The chain, explorer, full wallet, Popup wallet, staking DApps, and DEX path are operational. Pullup is broken, and local Windows 11 node binaries are not currently usable in the tested environment.
