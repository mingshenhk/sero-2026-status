This page collects confirmed or recurring issues affecting the SERO ecosystem in 2026.

## Status Labels

Use these labels when adding issues:

- **Unverified**
- **Partial**
- **Confirmed**
- **Resolved**
- **Outdated Guide**
- **Broken Link**
- **Needs Retest**

## Current Issues

### 1. Current infrastructure status is still incomplete
- Status: **Confirmed**
- Impact: Users do not yet have a single trustworthy public page showing what still works
- Notes: This repository exists partly to solve that problem

### 2. Historical documentation may be outdated
- Status: **Confirmed**
- Impact: Old setup steps may fail on modern systems
- Notes: Every old guide should be retested before reuse

### 3. Wallet usability is not yet fully verified
- Status: **Unverified**
- Impact: Users may not know whether fresh setup and sync still work
- Notes: Needs real installation and transfer tests

### 4. Staking flow is not yet fully verified
- Status: **Unverified**
- Impact: Users should not assume rewards or entry/exit flow works exactly as old documentation states
- Notes: Needs live end-to-end testing

### 5. Public explorer availability is not yet fully verified
- Status: **Unverified**
- Impact: Users may lack easy block and transaction visibility
- Notes: Needs comparison against node / RPC data

### 6. Exchange access may be highly limited
- Status: **Partial**
- Impact: Even if a market exists, real liquidity may be extremely weak
- Notes: Must distinguish between “listed” and “tradable in practice”

## How To Report A New Issue

When adding an issue, include:

- title
- date observed
- status
- affected component
- reproduction steps
- logs or screenshots if possible
- whether the issue is still current
