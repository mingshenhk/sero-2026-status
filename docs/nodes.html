Node and RPC Status

Mixed Last Updated: 2026-03-18

The chain is live and current, but local Windows 11 go-sero / gero node execution is broken in the tested environment.

Confirmed findings

  • Chain height appears current
  • Explorer height matches current chain state
  • Maintainer is actively mining on the current chain
  • Local sync target block count looked correct during testing
  • go-sero release downloads can be obtained

Confirmed failures

  • go-sero / gero binaries do not run successfully on the tested Windows 11 machine
  • gero.exe shows missing DLL error
  • Older versions were also tested and still failed on the same environment
